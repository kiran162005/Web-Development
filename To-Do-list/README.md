# To-Do List Web App

This is a simple To-Do List application built using HTML, CSS, and Vanilla JavaScript.
It allows users to add, complete, and delete tasks dynamically.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Instant UI updates

## How to Run
Open `index.html` in any web browser or use Live Server in VS Code.
